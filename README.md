# E-Commerce Sales Data Analysis Using SQL

This project analyzes an e-commerce sales dataset using SQL to extract meaningful business insights such as revenue analysis, customer behavior, and product performance.

---

## Dataset

The project uses four tables:

Customers  
customer_id  
customer_name  
city  

Products  
product_id  
product_name  
category  
price  

Orders  
order_id  
customer_id  
order_date  

Order Items  
order_id  
product_id  
quantity  

---

## SQL Questions Solved

1. Total revenue generated by each product category  
2. Customer who generated the highest revenue  
3. Top 3 most sold products  
4. Total orders placed by each customer  
5. Customers who never placed an order  
6. Order with highest revenue  
7. Cumulative revenue by order date  
8. Most expensive product in each category  
9. Latest order placed by each customer
